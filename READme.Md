# Healthcare Backend

This is the backend of the Healthcare Management System built with **Django** and **Django REST Framework**.

## Features

- User registration and login (Doctors & Patients)
- JWT Authentication with access and refresh tokens
- CRUD operations for Users, Doctors, and Patients
- Profile management
- API endpoints for frontend integration

## Tech Stack

- Python 3.x
- Django 4.x
- Django REST Framework
- PostgreSQL
- JWT Authentication (djangorestframework-simplejwt)
